Modelling of ΔR values around the coast of Australia Method
Sean Ulm, Damien O'Grady (CABAH, JCU Cairns)

Introduction

192 Records of ΔR measurements containing δ18O and δ13C values and Lat-Long coordinates were processed. The distribution of sites is shown in Fig. 1.

Method and results

The intention was to model the ΔR values on location and other parameters. Given that site locations form a near continuous loop around the coast, for the purposes of the model, the latitude and longitude were converted to polar coordinates about the centre of the continent, so that the bearing formed a continuous loop with a period of 360°.

Models

The purpose of the modelling was to create a tool to predict ΔR values at any point around the coast, in situations where both, one or neither of δ18O and δ13C were known. For this reason, four models were run, using:

  1. All parameters
  2. All except δ18O
  3. All except δ13C
  4. Location only

Conclusion

An R2 value of 0.65 for the full model was considered reasonable. Although it can be argued that a model using location as a predictor can always be manipulated to compliance, given sufficient complexity and a measure of over-fitting, the fact that varied data existed in the same locations that were distinguished by their δ18O and δ13C values alone, lends some weight to the observed R2 value. What should also be considered is the fact that the purpose of the exercise was to produce a tool to predict ΔR values within the domain of observed ΔR values. Model coefficients and variance-covariance matrices corresponding to each of the models were formed into javascript multi-dimensional arrays, which were configured to calculate a ΔR prediction with and without either of the δ18O and δ13C values, using the resultant models.

Note that the errors shown represent the standard error based on covariances within the described model alone, assuming zero measurement errors.

Actually the measured ΔR values had an average standard error of 60 years.
The measured δ13C values had an average standard error of 0.49.
The measured δ18O values had an average standard error of 0.28.

The errors associated with the measurement of ΔR (averaging ±60 years) are not included in the stated standard errors. It is generally accepted that measurement errors of the dependent variable do not affect the coefficients of a linear model, if it can be assumed that the measurement errors have an expected value of zero. Such errors may have had an adverse effect on the ability to establish the statistical significance of the model.
