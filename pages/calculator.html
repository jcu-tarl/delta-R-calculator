ΔR Calculator

Revised 2020-10-01

Radiocarbon (14C) ages obtained on contemporaneous terrestrial and marine samples are not directly comparable. ΔR values quantify regional deviations in marine 14C activity from the modelled global marine calibration curve (e.g. MARINE20). ΔR values can be used to calibrate radiocarbon ages more accurately using programs such as OxCal and CALIB.

How to use:

Enter Latitude and Longitude (or click on the map). If δ13C and/or δ18O are known, enter those, and click the button to calculate ΔR. An explanation of what you see may be found on the about page.

A zero value will influence the result. If no δ13C value available, empty the cell.
A zero value will influence the result. If no δ18O value available, empty the cell.
