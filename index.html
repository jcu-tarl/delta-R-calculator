ΔR Calculator

Enter Latitude and Longitude (or click on the map). If δ13C and/or δ18O are known, enter those, and click the button to calculate ΔR. An explanation of what you see may be found here Revised 2020-06-04
Longitude: Enter Decimal Longitude
Latitude: Enter Decimal Latitude
δ13C: Enter δ13C. Zero is a value which will influence the result. If no δ13C value available, empty the cell
δ18O: Enter δ18O. Zero is a value which will influence the result. If no δ18O value available, empty the cell
